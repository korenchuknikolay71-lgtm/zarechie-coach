# S&C

## Назначение проекта

Проект помогает тренеру S&C составлять, сохранять и анализировать тренировки для игроков. Данные берутся из Redis, тренировки и разминки генерируются через OpenAI Responses API и сохраняются обратно в Redis.

## Для кого создан

Проект создан для тренера, который работает с игроками волейбольной команды. В коде также есть рабочее пространство `nkperf`.

## Основные возможности

- Загрузка состава.
- Генерация индивидуальной тренировки на дату.
- Генерация разминки.
- Сохранение и загрузка тренировок.
- Учет 1RM, фактических весов и тоннажа.
- Учет ограничений, травм, самочувствия и истории сессий.
- Библиотека упражнений с изображениями, видео, категориями и дедупликацией.
- Командная готовность и расписание.
- Публичная карточка игрока по токену.

## Поток данных (от спортсмена до тренера)

1. Данные спортсмена лежат в Redis: WHOOP, опросники, утренние чек-ины, нейротесты, травмы и ручные данные.
2. API собирает снимок игрока через `getPlayerSnapshot`.
3. Генератор добавляет историю сессий, 1RM, ограничения, обратную связь, расписание и командный контекст.
4. OpenAI Responses API возвращает структурированную тренировку.
5. Тренер видит тренировку в интерфейсе, может сохранить ее и внести фактические данные.
6. Сохраненные данные попадают обратно в Redis и используются в следующих расчетах.

## Что уже реализовано

- Авторизация через `TRAINER_API_KEY`.
- Подключение к Redis через `KV_REST_API_URL` и `KV_REST_API_TOKEN`.
- Индивидуальная генерация тренировок через OpenAI.
- Генерация разминок через OpenAI.
- Batch-подобный поток генерации через `generate-async` и `generate-status` без изменения интерфейса.
- Валидация сессий после генерации.
- Сохранение версий сессии.
- Расчет тоннажа и gym-ACWR.
- Библиотека упражнений с Redis-схемой `ex:*`.
- Warmup-история.
- Team readiness.
- Интеграция NK Performance roster.

## Что находится в разработке

- В рабочем дереве есть незакоммиченные новые и измененные файлы.
- По коду видны активные зоны: team endpoints, warmup, schedule, NK Performance, forecast, forceplate, actual weights, exercise memory, team playbook.
- Отдельного файла с дорожной картой в проекте нет.
